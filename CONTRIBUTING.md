# Contributing

Thanks for helping improve Alzheimer's CareKit.

## Product guardrails

- Keep the project focused on caregiver organization and handoff support.
- Do not add diagnosis, treatment, clinical scoring, or emergency decision features.
- Prefer local-first behavior for private care data.
- Make accessibility part of every change.
- Keep language plain and non-alarming.

## Development

```bash
npm install
npm run lint
npm run build
```

## Pull requests

Please include:

- A short summary of the user-facing change
- Any privacy or safety impact
- Screenshots for UI changes
- Manual test notes
